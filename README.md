# Dotconfig

My personal `.config`

## Requirements

1. [Mise](https://mise.jdx.dev/)

## Setup

```
$ git clone https://github.com/iamchanii/dotconfig ~/.config
$ mise install
$ ln -s ~/.config/.zshrc ~/.zshrc
```

## Link Claude Code Slash Commands

```
$ rm -rf ~/.claude/commands
$ ln -s ~/.config/claude/commands ~/.claude/commands
```
